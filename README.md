# Library Management System (Java Application)

This is a foundational Java application designed to simulate the core functions of a library. It is structured to demonstrate a clean, maintainable approach to solving a real-world business problem.

The core objective of this project is to showcase clean code and professional structure:

1.  **Object-Oriented Design (OOD):** Clean separation of responsibilities into core entity classes (`Book`, `Member`) and service classes (`LibraryService`).
2.  **Standard Structure:** Utilizes the standard `src/main/java/com.library` package structure for enterprise readiness.
3.  **Data Structures:** Employs Java Collections (e.g., `ArrayList`, `HashMap`) for managing in-memory data persistence.
